## API Basics

Before you begin!
You should create a free [KidaPay account]() that you can test the API against. We will provide you with test keys and wallets that you can use to make API calls and payments.



The KidaPay API gives you access to pretty much all the features you can use on our merchants app and lets you integrate them for use in your application. It strives to be RESTful and is organized around the main resources you would be interacting with - with a few notable exceptions.
